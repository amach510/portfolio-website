Meet

This Progressive Web App (PWA), constructed through Test-Driven Development (TTD), allows users to log in via OAuth2.0 using their Google account to view a roster of events. Users can filter the list by location (city) and specify the number of events to be shown. The application employs serverless functions (AWS Lambda) to acquire the access token from the authorization server and to retrieve the event list from the Google Calendar API.

Tech used: React, React-bootstrap, Rechart, OAuth2.0, serverless functions, AWS Lambda
